# SOJO Calculator (Android/Kotlin)

This project is a simple calculator with some extra features developed for Android in Kotlin.

![Screenshot 2026-05-10 205938](https://github.com/sorjoh/SOJO-Portfolio/blob/main/SOJO%20Calculator/Android.Kotlin/Calculator.png)

## Features
- **Basic Mathematical Operations:** Addition, subtraction, multiplication, and division.
- **Extra Functions:** Percentage calculation, square and cube root, power, pi, modulus, reciprocal and random number generator.
